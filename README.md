# Taskify

Taskify is a simple, drag-and-drop to-do list web app built with **HTML, CSS, and JavaScript**. Organize your tasks efficiently across columns and keep track of progress with automatic task counts.  

## Features

- **Drag & Drop Tasks** – move tasks across columns seamlessly  
- **Automatic Task Counting** – keep track of your progress  
- **Responsive Design** – works on both desktop and mobile  
- **Installable as a Web App** – thanks to `manifest.json`, you can add Taskify to your home screen
